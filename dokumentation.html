Dokumentation

Meny

Längst upp till höger på alla html sidor finns en responsiv meny som vid upplösningar under 758 pixlar ändras till en hamburgarmeny.
Meny är även animerad för att ge en snygg effekt av en växande bubbla som växer.

index.html

Innehåller canvas för att kunna rita på. Nedan finns en toolbar som innehåller färger man kan använda.
Varje färgknapp har en javascript eventlistener knyten till sig som hämtar bakgrundsfärgen på elementet och knyter denna färgen till color variablens som används för att bvestämma färgen på stroke.

app.js

Innehåller javascript koden för att kunna rita på index.html.
Dokumentet innehåller kommenterad kod som förklarar vad varje funktion gör.

meny.js

Innehåller javascript koden för att få den responsiva hamburgarmenyn att fungera.
Dokumentet innehåller kommenterad kod som förklarar vad varje funktion gör.

style.css

Innehåller all CSS koden för att få hemsidan att se snygg ut.
